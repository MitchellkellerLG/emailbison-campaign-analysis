üîç Critical Insights from TeachAid Analysis

1. Template Consistency

  • TeachAid uses ONE core template across 90% of campaigns
  • The "Peel District pilot" message is their go-to approach
  • Only variations are in Spintax, not core messaging

2. Geographic Strategy

  • Ontario (Peel): Most developed campaigns, active outreach
  • Manitoba: Created but incomplete - abandoned?
  • British Columbia: Empty campaigns - never launched
  • USA (BOCES): Incomplete - different approach needed?

3. Winning Elements

  • "peel district pilot" - Local credibility
  • "over coffee instead of a weekend" - Time-saving visual
  • "on the house" - Free trial removes friction
  • "pedagogically aligned" - Education terminology
  • Heavy {COMPANY} personalization - School-specific

4. A/B Testing Pattern

  • Main variant: Professional tone, detailed explanation
  • Test variant: Casual "crazy idea" opener, shorter
  • Both use same core elements (Peel pilot, free, time-saving)

5. Missed Opportunities

  • 17 of 25 campaigns are incomplete or empty
  • No US-specific messaging developed
  • No follow-up sequences visible in most campaigns
  • Could expand beyond Peel District reference for other regions

üìà Recommendations for TeachAid

1. Complete Unfinished Campaigns

17 campaigns have incomplete or missing templates - huge untapped potential

2. Develop Regional Variations

  • Manitoba: Reference local school divisions instead of Peel
  • BC: Mention BC curriculum alignment
  • USA: Adapt for American terminology (districts vs boards)

3. Expand Subject Lines

Test variations beyond "peel district pilot":

  • "2026 lesson plans"
  • "teacher time-saver"
  • "free AI planning tool"
  • "[district name] pilot opportunity"

4. Add Follow-Up Sequences

Most campaigns have only 1-2 steps. Add:

  • Teacher testimonials
  • Case study examples
  • Time-saving calculations
  • Implementation timeline

5. Target IT Decision Makers

1,373 IT personnel identified but using same teacher-focused message. Create IT-specific angle focusing on:

  • Security/privacy compliance
  • Integration with existing systems
  • Deployment simplicity
