## Key Insights & Patterns

### What's Working:
1. **Reddit positioning** - Timely with Google-Reddit partnership
2. **Free value offers** - Reports and audits vs. direct pitches
3. **Casual, conversational tone** - "coulda," "lmk," informal language
4. **Specific metrics** - "10-30k visitors," "60k monthly"
5. **White-label opportunities** - Strong angle for agencies
6. **Multi-step sequences** - Building value across 3-4 emails

### What's Failing:
1. **Financial services messaging** - 0% interest rate
2. **Generic outreach** - Without specific value props
3. **Single-email campaigns** - All winners use sequences
4. **Overly formal tone** - Best performers use casual language
